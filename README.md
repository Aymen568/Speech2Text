# استوديو الصوت

تجربة صفحة واحدة بـ HTML/CSS/JS لتسجيل الصوت، تحويله إلى نص، وعرض أسئلة ذات صلة في شريط جانبي أيمن. يدعم بثًا حيًا للنصوص من الخادم عبر SSE.

## كيفية التشغيل
1. الواجهة: شغّل مجلد المشروع عبر خادم ملفات ثابت (مثل `npx serve .` أو إضافة Live Server في VS Code). يحتاج الميكروفون إلى `https` أو `localhost` للسماح.
2. افتح ملف `index.html` في المتصفح.
3. الخادم: من مجلد `Speech2Text` ثبّت المتطلبات ثم شغّل FastAPI عبر uvicorn:
	- تثبيت: `pip install fastapi uvicorn speechmatics httpx`
	- تشغيل: `uvicorn main:app --reload --port 8000`
	- ضع مفتاحك في المتغير `SPEECHMATICS_API_KEY`.

## طريقة الاستخدام
- اضغط **ابدأ التسجيل** لالتقاط الصوت عبر `MediaRecorder` (سيُطلب الإذن).
- اضغط **إيقاف** لإنهاء التسجيل؛ واجهة وهمية تحوّل الصوت إلى نص وتعرضه مباشرة.
- يتم تحديث الأسئلة تلقائيًا؛ استخدم **تحديث** لإعادة التوليد يدويًا.
- زر **بث الخادم** يتصل بنقطة `/stream` في الخادم ويعرض النصوص المتدفقة (جزئية ونهائية).
- زر **مسح النص** يزيل النص والأسئلة.

## ملاحظات
- الدالة `mockTranscribeAudio` في `main.js` تحاكي طلب خادم؛ استبدلها بطلبك الحقيقي وأعد النص.
- الصوت يبقى داخل المتصفح؛ الإرسال للخادم يحدث فقط إذا ربطت نقطة نهاية حقيقية.
- بث الخادم يعتمد على SSE من `http://localhost:8000/stream`؛ عدّل العنوان حسب نشر الخادم.
